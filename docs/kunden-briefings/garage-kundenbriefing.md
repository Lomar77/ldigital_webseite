# Ausführliches Projektbriefing für Garage-Webseiten

Vielen Dank für Ihr Interesse. Mit diesem Fragebogen sammeln wir die wichtigsten Informationen für Ihre neue Webseite.

Das Briefing dient als Grundlage für eine realistische Offerte und saubere Umsetzung. Je genauer die Angaben sind, desto besser können Aufwand, Preis und Umfang eingeschätzt werden. Offene Punkte können im Gespräch geklärt werden.

Bitte beantworten Sie die Fragen so gut wie möglich. Wenn Sie bei einem Punkt noch unsicher sind, schreiben Sie einfach „noch offen“. Gemeinsam klären wir anschliessend die fehlenden Angaben.

Mit **Pflicht** gekennzeichnete Angaben werden vor Projektbeginn benötigt. Alle anderen Angaben sind optional.
