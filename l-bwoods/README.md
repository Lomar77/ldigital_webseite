# L-BWoods

L-BWoods ist das interne Finanz- und Unternehmenssystem von L-Digital.

Das Projekt startet bewusst als schlanker, dokumentierter Prototyp. In Version 1.0 werden Finanzbewegungen, Belege, Assets, Projekte und wichtige Meilensteine mit CSV-Dateien und Markdown-Dokumentation erfasst. Es ist keine fertige Web-App und keine vollstaendige Buchhaltungssoftware.

L-BWoods existiert, um den Aufbau von L-Digital sichtbar und nachvollziehbar zu machen:

- Kapitalbewegungen
- Einnahmen
- Ausgaben
- Belege
- Assets
- Projekte
- Journal / Meilensteine

Sensible Finanzdaten und echte Belege muessen geschuetzt werden. Sobald echte Finanzdaten, Bankdaten oder Belegbilder in diesem Projekt gespeichert werden, muss das Repository privat bleiben.

Echte Belegbilder werden standardmaessig durch `.gitignore` ausgeschlossen und duerfen nicht versehentlich oeffentlich versioniert werden.
