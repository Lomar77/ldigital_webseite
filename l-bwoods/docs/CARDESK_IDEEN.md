# CarDesk Ideen

Teile der L-BWoods-Logik koennen spaeter fuer CarDesk interessant werden, vor allem fuer einfache Auswertungen und Management-Uebersichten.

Moegliche Kennzahlen und Ansichten:

- Umsatz pro Kunde
- Umsatz pro Fahrzeug
- Umsatz pro Auftrag
- offene Rechnungen
- offene Offerten
- Materialkosten pro Auftrag
- Geschaeftsfuehrer-Dashboard
- Werkzeug-Investitionen

CarDesk soll dadurch keine vollstaendige Buchhaltung werden. Die Idee ist eine klare operative Uebersicht fuer Kunden, Auftraege, Material und Investitionen.
